### 网上花店管理系统✔万字文档(LW)✔Java✔SSM✔Vue✔启动教程
## 源码 + 万字文档（LW）+  SQL脚本 + 软件环境安装包
## 【项目采用技术】
架构: B/S、MVC
系统环境：Windows/Mac
开发环境：IDEA、JDK1.8、Maven、MySQL、tomcat7.5/8/9
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、Vue、JavaScript、CSS、Javaweb
## 【系统功能模块】
本系统包含管理员和用户2个角色。
①管理员角色：订单信息管理，用户管理，鲜花管理，鲜花类型管理，鲜花订单管理、配送管理、审核管理
②用户角色：登录、注册、修改密码、首页查看鲜花信息、留言、下单、评价、个人中心
